Разведывательный анализ данных (EDA):


основные цели и задачи проекта:

отследить влияние условий жизни учащихся в возрасте от 15 до 22 лет на их успеваемость по математике, чтобы на ранней стадии выявлять студентов, 
находящихся в группе риска, с помощью модели, которая предсказывала бы результаты госэкзамена по математике для каждого ученика

краткая информация о данных:

в файле для анализа представлена информация о родителях, образовании и месте работы; месте жительства; оценки по математике; дополнительных занятиях и т.п студентов  


этапы работы над проектом, основные шаги:

Первичный осмотр данных;
Проверка количества уникальных значений;
Проверка данных на пустые значения;
Проверка данных на наличие выбросов;
Корреляционный анализ количественных переменных;
Отобратка данных, пригодных для дальнейшего построения модели;
Анализ номинативных переменных и устранение тех, которые не влияют на предсказываемую величину (в нашем случае — на переменную score);
